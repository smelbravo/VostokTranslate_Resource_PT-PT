## 🌍 We Need Native Speakers!

We are looking for native speakers to improve translation quality.

Even small fixes are welcome!

## 🤝 How to Contribute (Translation)

1. Fork this repository
2. Edit ONLY files in `supplemental/`
3. Add translations in this format:

```json
{
  "Health": "体力"
}
```

4. Do NOT edit generated files:

* lang/*.json
* config/*.json

5. Submit a Pull Request
